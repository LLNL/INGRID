class geometry.Line(points)¶

Line object, in which an ordered set of points defines a line.

Parameters:points (list) – Points are of the form p = (x, y), and the list should be made up of multiple points. [p, p, p]…
divisions(num)¶

Splits the line into discrete segments.

Parameters:num (int) – Number of points in the segmented line.
plot(color='#1f77b4')¶

Plots the line of the current figure.

Parameters:color (str, optional) – Defaults to a light blue.
points()¶

Returns the points in the line as a tuple.

print_points()¶

Prints each point in the line to the terminal.

reverse()¶

Points the line in the other direction. It is intended to be used right after generating a line using the draw_line funciton from the line tracer. For example; LineTracing.draw_line(args…).reverse().

Returns:geometry.Line
Return type:self
class geometry.Patch(lines)¶

Each patch contains a grid, and has it’s own shape.

Parameters:lines (geometry.Line) – Each patch defined by four lines in order - Nline, Eline, Sline, Wline - order points to go clockwise.
fill(color='lightsalmon')¶

Shades in the patch with a given color

Parameters:color (str, optional) – Defaults to a light salmon.
plot_border(color='red')¶

Draw solid borders around the patch.

Parameters:color (str, optional) – Defaults to red.
refine(grid)¶

Divides a patch into smaller cells based on N and S lines, and the psi levels of E and W lines.

Parameters:grid (Setup_Grid_Data.Efit_Data) – Requires the grid the patches were calculated on.
