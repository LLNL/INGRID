Ingrid Class¶

Created on Fri Jun 21 15:17:21 2019

class Ingrid.Ingrid(nml)¶

An interactive grid generator for edge plasmas in a tokamak Accepts a dictionary generated from a namelist file that contains the parameters.

Parameters:nml (dict) – Params dictionary object contains two dictionaries First is ‘files’ which contains the keys: geqdsk, itp, otp. The second is ‘grid params’ which has: psi_max, psi_min_core, psi_min_pf, Rmagx, Zmagx, Rxpt, Zxpt
OMFIT_read_psi()¶

Python class to read the psi data in from an ascii file. Saves the boundary information and generated efit_data instance

add_magx(r=None, z=None)¶

Adds the magnetic axis using the coordinates of last point that the user has clicked.

add_xpt1(r=None, z=None)¶

Adds the X-point using the coordinates of last point that the user has clicked.

calc_efit_derivs()¶

Calculates the partial derivatives using finite differences. Wrapper for the member function in the efit class.

calc_psinorm()¶

Uses magx and xpt1 to normalize the psi data. Furthur calculations will use this information

compute_eq_psi()¶

Initializes the line tracing class for the construction of the grid. Necessary to be separate so we can wiat until the user clicks on the root.

construct_SNL_patches()¶

More general format to construct the grid for SNL using patches. For theta direction, cw is positive. For rho direction, ccw is positive, which is away from the magx.

Lines are drawn in the form: self.eq.draw_line(, , option=’‘, direction=’‘)

Patch Labeling Key:
I: Inner, O: Outer, DL: Divertor Leg, PF: Private Flux, T: Top, B: Bottom, S: Scrape Off Layer, C: Core.
draw_polodal_lines()¶

Trace contour lines anywhere you click saves the most recent line that was draw. To keep track of lines, call the add_line function

draw_radial_lines()¶

Trace the orthogonal lines to the stream function saves most recent line.

export()¶

Saves the grid as an ascii file

find_roots()¶

Displays a plot, and has the user click on an approximate zero point. Uses a root finder to adjust to the more exact point. Right click to disable.

get_param(key=None)¶

Returns the value associated with a key.

import_psi_data()¶

Same as OMFIT_read_psi, but calls a fortran function to read in the psi data. Saves and creates the same instance.

patch_diagram()¶

Generates the patch diagram for a given configuration.

plot_efit_data()¶

Generates the plot that we will be able to manipulate using the root finder

plot_target_plate()¶

Plots the inner and outer target plates on the current figure

print_params()¶

Lists the parameters that the user should know about.

read_target_plate()¶

Reads the coordinates for a line defining the inner and outer target plates. The lines to define target plates end up with the shape ((x,y),(x,y)). These files read can contain more complicated lines than this.

refine_patches()¶

Break each patch into smaller grids based on psi.

save_root()¶

Returns the value of the root finder. This will either be the root, or the click, depending on the root finding settings.

set_param(key=None, value=None)¶

Sets the parameters that we want, without adding new and unwanted options. User can pass in new values directly in, or can be prompted.

Parameters:
  • key (str, optional) – Keyword for the dictionary object. Specify which value we wish to change. Will be prompted for it if is None.
  • value (varies, optional) – Value associated with a key. Must match the type of the value currently tied to the key.
setup()¶

Add the magnetic axis and the x-point

test_interpol(option=2, nfine=100, ncrude=10, tag='v')¶

Provides a demonstration and test of the bicubic interpolation methods used. Wrapper for the real code from another module.

Parameters:
  • option (int, optional) – Specify which of a set of function to use as the test. Accepts 1, 11, 12, 13, 2, 3, 4, 5, 51, 52 See Test_Functions.get_f for more detail.
  • nfine (int, optional) – Density of the fine grid we will interpolate onto.
  • ncrude (int, optional) – Density of the crude grid the sample data will be generated onto.
  • tag (str, optional) – Specify if it is wanted to test the derivative interpolation methods. Accepts ‘v’, ‘vr’, ‘vz’, ‘vrz’.
toggle_root_finder()¶

Activates or deactivates the root finder ability. Enables the user to save the location where they last clicked.

Ingrid.run()¶

Reads a namelist file containing the parameters for Ingrid, and runs the grid generator for a single null configuration.

Ingrid.set_params()¶

Interactive mode for Ingrid. Gets the files used, and opens a plot of the Efit data. Prompts the user for magx, xpt, and psi levels. Saves the data in a namelist file.
