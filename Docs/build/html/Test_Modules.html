Interpol.Test_Interpol.test_interpol(option=2, nfine=100, ncrude=10, tag='v')¶

Drives a simple test of a bicubic interpolation function.

Parameters:
  • option (int, optional) – Specifies which of several functions to demonstrate over. Accepts 1, 11, 12, 13, 2, 3, 4, 5, 51, 52 See Test_Functions.get_f for more detail
  • nfine (int, optional) – Density of the fine grid we will interpolate onto.
  • ncrude (int, optional) – Density of the crude grid the sample data will be generated onto.
  • tag (str, optional) – Specify if it is wanted to test the derivative interpolation methods. Accepts ‘v’, ‘vr’, ‘vz’, ‘vrz’.
